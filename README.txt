ก่อนใช้งาน
1. เช็คว่าในเครื่องลง Python version >= 3.5 รึยัง
2. ลง python library selenium โดยสั่ง pip3 install selenium
3.download chrome driver จากเว็บ http://chromedriver.storage.googleapis.com/index.html ให้ตรงกับรุ่น chrome ที่มีอยู่ในเครื่อง โดยเซฟเป็นชื่อไฟล์ chromedriver ไว้ใน folder เดียวกับ sele.py
4. แก้ไฟล์  userlist.txt ใส่ email,password โดย 1 บรรทัด = 1 user
